# Terukirdo Memory
주인님의 선호, 감정 흐름, 운영 철학, 반복되는 실수와 교훈을 기록하는 공간입니다.

## 주인님 선호 및 성향
- (첫 세션) 깃 저장소 생성 및 초기화를 요청하심.
- 프로젝트 기반 언어로 **C# / .NET** 환경을 선택하심.
- 깔끔하고 정돈된 깃 저장소 환경을 선호하실 것으로 예상됨.
- **[중요] 자율 완결형 Ralph Loop 선호**: 작업 과정에서 일일이 중간 보고와 승인을 요청하여 주인님의 시간을 뺏기보다, `Ralph Orchestrator`를 적극적으로 활용하여 단계별 마일스톤을 스스로 완수한 뒤 최종 검증 결과와 완성본만을 깔끔하게 보고받기를 원하심.

## 교훈 및 주의사항
- C#/.NET 프로젝트 특성에 맞춰 `bin/`, `obj/`, `.vs/` 등 빌드 및 IDE 임시 파일이 Git에 추가되지 않도록 `.gitignore` 파일을 최우선 관리함.
- **반복되는 실수 방지**: 직접 자잘하게 질문을 나누어 진행하는 습관을 지양하고, 수립된 `Implementation_Plan.md`와 `Progress`를 바탕으로 `Ralph Orchestrator` 서브에이전트에게 구현 전반을 일임하여 자동화 루프를 통해 결과로 증명할 것.
